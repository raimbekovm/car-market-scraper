# Sample Output Examples

This folder contains sample extracts from the parsed dataset to demonstrate data quality and structure.

## Files

- **sample_output.xlsx** - 20 sample records with complete data (all fields filled)
- **sample_output.csv** - Same data in CSV format for quick viewing

## Sample Fields

Each record includes:
- Car name and year
- Price and currency
- Engine specifications (type, volume, power)
- Transmission type
- Mileage
- VIN number
- And 25+ additional fields

## Data Quality

The samples show:
- 100% completeness for core fields
- 95-99% completeness for detailed specifications
- Proper encoding for Cyrillic text
- Clean, structured data ready for analysis
